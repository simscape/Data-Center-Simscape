Overview

This block implements a grid-forming control strategy for Battery Energy Storage Systems (BESS). Unlike grid-following controllers, this block establishes voltage and frequency references for the grid or microgrid, enabling the BESS to operate as a voltage source.
You can operate this block in two selectable modes:
  • Droop Control Mode: Provides decentralized power sharing based on frequency-power and voltage-reactive power droop characteristics.
  • Virtual Synchronous Machine (VSM) Mode: Emulates synchronous machine dynamics, improving system inertia and stability.
You can use this block for :
  • Microgrid Operation: Black-start capability and voltage/frequency reference.
  • Renewable Integration: Provides inertia and damping for stability.
  • Grid Services: Frequency regulation, voltage support, and islanded operation.

Detailed Description

Block Functionality

The controller generates voltage and current references for the BESS converter. It ensures:
  • Voltage and frequency regulation.
  • Support for black-start and islanded operation.

Control Structure

The following table gives you the information about different units in the controller
Unit
Description
Droop Control
Implements P–f and Q–V droop for decentralized power sharing. Generates voltage reference based on power measurements and droop coefficients.
Virtual Synchronous Machine
Emulates synchronous machine dynamics for inertia support and damping for frequency stability
Voltage Controller
Regulates converter terminal voltage in dq-frame
Current Controller
Generates dq current references for converter
This block supports two operational modes:
  • VF Mode: Regulates terminal voltage magnitude and frequency, typically via droop control or virtual synchronous machine (VSM) emulation.
  • PQ Mode: Regulates active power (P) and reactive power (Q) delivered to the grid while maintaining stable internal voltage and current operation.
You can choose either of these modes using the Mode input to the controller. Provide input :
  • 0 for VF mode
  • 1 for PQ mode
