Overview

This block models a double conversion UPS system that provides uninterrupted and regulated AC power to a critical load. The UPS continuously processes power through a rectifier–DC link–inverter chain, ensuring complete electrical isolation from upstream disturbances.
The UPS operates on the principle of continuous power conversion:
  • Input AC power is converted to DC by the rectifier
  • The DC link stabilizes energy flow and interfaces with the battery
  • The inverter regenerates a controlled AC waveform to supply the load
The battery supports the DC link to ensure uninterrupted operation during grid disturbances or outages.
The UPS supports the following operating states:
Normal Operation
  • Grid feeds the rectifier
  • Rectifier maintains DC-link voltage
  • Inverter supplies regulated AC power to the load
  • Battery remains in float or controlled charging mode
Battery Operation
  • Triggered when grid voltage is unavailable or out of range
  • Battery supplies power to the DC link
  • Inverter continues to supply uninterrupted AC power

Detailed Model

Block Description

The detailed model consists of three main components:
  • Rectifier
  • DC-Link with Battery
  • Inverter

Rectifier

You can configure the rectifier block to match your desired rating by specifying parameters such as input voltage, current limits, switching frequency, and control gains. This block models the AC–DC conversion stage and regulates the input current drawn from the grid.
This block implements an power factor correction (PFC) control strategy, which shapes the input current to be:
  • Sinusoidal
  • In phase with the grid voltage
This ensures that the UPS operates with a near-unity power factor and minimizes harmonic distortion injected into the grid.
This block simulates:
  • Conversion of three-phase AC input to regulated DC output
  • Input current control using PFC
  • DC-link voltage support through controlled power flow
  • Interaction with upstream grid through voltage and current feedback

DC-Link and Battery

The DC-link subsystem represents the energy buffer between the rectifier and inverter. It includes a DC bus capacitor and a battery system for energy storage and backup operation.
This block simulates:
  • DC voltage dynamics and energy storage
  • Battery charging and discharging behavior
  • Power balance between rectifier and inverter
Control inputs such as charging current, rectifier current, and voltage feedback are used to regulate the DC-link operation under different conditions.

Inverter

The inverter converts DC-link voltage into a regulated AC output that supplies the load. It ensures continuous and high-quality power delivery independent of input disturbances.
This block simulates
  • DC–AC conversion
  • Output voltage regulation
  • Frequency and waveform control
  • Load supply under both normal and battery operation

Key Characteristics

  • Continuous double conversion operation (zero transfer time)
  • Full decoupling of load from grid disturbances
  • Integrated battery energy storage through DC link
  • Measurement-enabled architecture for system monitoring
  • Suitable for data center and critical power applications
