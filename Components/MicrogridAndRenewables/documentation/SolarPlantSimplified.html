Overview

This block models a solar plant using Anderson's translational equations which operates in grid-following mode. You can use this block to abstract the solar plant behavior using Anderson’s translational equations. It requires only typical datasheet values as inputs, such as :
  • Open-circuit voltage ()
  • Short-circuit current ()
  • Maximum power point voltage/current (, )
  • Temperature coefficients (α, β)

Anderon’s equations

Anderson’s translational equations are a set of empirical formulas that estimate the electrical output characteristics (current, voltage, power) of a photovoltaic (PV) module or array under varying environmental conditions. They provide a simplified yet effective way to predict PV performance based on standard test condition (STC) data sheet values, irradiance, and temperature, without requiring a full physical or equivalent circuit model.
You can use these equations when computational speed is critical, such as in large-scale grid studies or for preliminary PV system sizing.

Functionality

This block assumes ideal or near-ideal tracking and conversion, neglecting dynamic effects and high-frequency behaviors. This block operates in two modes namely Maximum Power Point Tracking (MPPT) and Power reference modes. If you choose the Operating Mode option on the block to Power Reference external inputs for active and reactive power references would appear. You can choose either of these modes according to your requirements. This block also limits the output current to the value defined by Maximum current of the converter parameter on the block mask
This block allows you to select the type of solar panel from the mask parameters. When you choose Polycrystalline or Monocrystalline as the panel type, the temperature correction factor for is automatically set to 0.111 and 0.063, respectively. If you select Cadmium Telluride or Other panel types, an additional option appears on the mask, allowing you to manually set the temperature correction factor. For Cadmium Telluride, the default value is 0.111, but you can modify this value to suit your specific requirements. This flexible configuration ensures accurate modeling for different solar panel technologies.
